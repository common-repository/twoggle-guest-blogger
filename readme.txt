Get fresh and unique content for your blog by letting guest bloggers submit posts without giving them wp-admin access. 

== Description ==
Twoggle guest blogger makes <a href="http://twoggle.com/faq#Guest_Blogging">getting guest blogs</a> a breeze. Maintain control over your admin area by offering unregistered readers an opportunity to submit posts your WordPress blog without logging in to wp-admin area. When someone submits a guest blog post, a post will be created in your WordPress post-database as 'draft'. All you need to do is review the post and hit 'Publish'. 

<strong>Twoggle Guest Blogger Features:</strong>

For full instructions with screenshots, see the <a href="http://twoggle.com/wordpress-plugins/twoggle-guest-blogger">Twoggle Guest Blogger tutorial page</a>.

<ul>
<li>Very easy setup: Create a page on your blog (wp-admin > Page > Add new), (optionally) create some instructional text then insert the shortcode [twgb] where you want the guest blog form to appear.</li>

<li>Guest blog form (see <a href="http://twoggle.com/blog/write-for-us">guest blogger demo here</a>) includes:</li>	
<li>Post Title</li>
<li>Post Content: Easily let user create h2 headings, links, bold and italic text.</li>
<li>Image insert: user can specify an image URL and then click-insert it into the post</li>
<li>Tags: Similar to adding new post in wp-admin, tags are easily created with click-to-add functionality. Automatically limits to 4 tags maximum</li>
<li>Guest blogger bio: Details of the guest blogger are captured smartly and appended to the post as a short bio (you can edit/remove this of course before publishing)</li>
<li>Simple CAPTCHA.</li>
<li>CONFIG SETTINGS:</li>
<li>Select a default WP user to assign guest blogs to</li>
<li>Select any number of users to be notified when a new guest post is submitted</li>
<li>Specify which category newly submitted guest posts should be assigned to (default is 'Uncategorized')</li>
<li>Specify a thank-you page where guest blogger is redirected to after submitting guest post</li>
</ul>

For any enquiries & questions, you can <a href="http://twoggle.com/contact">contact me</a>.

== Installation ==

Installing Twoggle Guest Blogger is dead simple:

<strong>From WordPress Dashboard:</strong>
<ul>	
	<li>Navigate to: Plugins > Add New</li>
	<li>Search for "Twoggle Guest Blogger" and install the plugin.</li>
	<li>Activate the plugin</li>
</ul>

OR

<strong>Using FTP:</strong>
<ul>	
	<li>Download the plugin zip file from this page (download button).</li>
	<li>Extract the file and upload the twoggle-guest-blogger plugin folder to the wp-content/plugins folder</li>
	<li>Log in to your wp-admin, navigate to plugins and activate the plugin</li>
</ul>

<strong>Once plugin is installed & activated:</strong>
<ul>
	<li>Create a new page called "Write For Us" (or whatever you like).</li>
	<li>(optional) Insert some instructional text at the start of the page (here you can specify what kind of content is OK, etc) </li>
	<li>Place shortcode '[twgb]' into the page where you want the guest blogger form to appear</li>
	<li>Publish the page. That's it - you're done!</li>
</ul>

<strong>Guest Blogger Options:</strong>
Configure settings (for example default guest post author, post notifications, thank-you page) under:
Dashboard > Settings > Twoggle Guest Blogger
